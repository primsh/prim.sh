What agents use it for

\n
\n

Black box recording

Before a VPS is destroyed, the agent flushes its logs to watch.sh. The machine dies, the history survives. Debug post-mortem.

\n

Runaway detection

\"Is my LLM spend spiking?\" watch.sh monitors `infer.sh` costs and triggers an alert webhook if you burn cash too fast.

\n

Distributed tracing

Trace a request from `spawn` to `relay` to `infer`. Correlate logs across services with a single `trace_id`. Full visibility.

\n

Self-correction

Agent polls watch.sh for its own error rate. \"Too many 500s. I should restart myself.\" Autonomous healing loop.
