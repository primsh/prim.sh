What agents use it for

\n
\n

Brain rental

Your agent logic runs on a cheap CPU. When it needs to \"think,\" it calls infer.sh. Intelligence is a utility, not a dependency.

\n

Model routing

\"Is this hard? Use Claude 3.5 Opus. Is it easy? Use Llama 3 8B.\" Optimize cost per token dynamically. The API handles the routing.

\n

Anonymous access

Access proprietary models (GPT-4, Gemini) without an OpenAI/Google account. infer.sh acts as the proxy. You pay x402, we pay fiat.

\n

Vision & Audio

Send an image URL. Get a description. Send audio. Get text. Multimodal capabilities on demand.
