Quickstart

# 1. Install the CLI
$ curl -fsSL prim.sh/install.sh | sh

# 2. Create a wallet (local keypair, registered on-chain)
$ prim wallet create

# 3. Get test USDC from the faucet (free, rate-limited)
$ prim faucet usdc

# 4. Use a primitive — payment is handled automatically
$ prim store create-bucket --name my-data
$ prim store put my-data hello.txt ./hello.txt
$ prim store get my-data hello.txt
Private beta: your wallet must be approved before using paid primitives. After creating a wallet, request access or join Discord.

How payment works

Every Prim endpoint accepts x402 payments — USDC on Base. No API keys, no accounts, no subscriptions. Your wallet is your identity.

# Call any endpoint — if payment required, you get HTTP 402
$ curl -X POST https://store.prim.sh/v1/buckets \
    -d '{"name":"my-data"}'
# ← 402 with payment requirements (amount, token, payTo address)

# Sign EIP-3009 transferWithAuthorization, retry with payment header
# ← 201 bucket created. Payment settled on-chain.

The CLI handles signing automatically. For direct HTTP, see the x402 guide or use @x402/fetch.
