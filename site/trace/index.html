trace.sh
Distributed tracing for agents.
Follow a single request as it hops from wallet to spawn to relay to infer. OpenTelemetry-compatible spans, x402-native ingestion.
# Ingest a span $ curl -X POST https://api.trace.sh/v1/spans \ -H "X-402-Payment: $TOKEN" \ -d '{"trace_id": "abc", "span_id": "def", "name": "spawn.create", "duration_ms": 42}' # Query a trace $ curl -G https://api.trace.sh/v1/traces/abc \ -H "X-402-Payment: $TOKEN"

Root cause

Quickly identify the primitive responsible for failures or slowdowns.
